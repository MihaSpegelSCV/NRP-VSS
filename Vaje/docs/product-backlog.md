**Product Backlog – Shieldy**

**Persona**

Ne-tehnični uporabnik interneta, ki želi enostavno zaščito pred phishing napadi.



**User Stories**

**🔴 MUST**

### 1

As a user,
I want to scan a suspicious link,
so that I know if it is safe.

Acceptance Criteria:

* User can input URL
* System shows SAFE or DANGER

---

### 2

As a user,
I want to get a clear warning if a link is dangerous,
so that I avoid scams.

Acceptance Criteria:

* Red warning is shown
* Message is clear

---

### 3

As a user,
I want to understand why something is dangerous,
so that I can learn.

Acceptance Criteria:

* Explanation is displayed
* At least one reason shown

---

### 4

As a user,
I want step-by-step instructions after detection,
so that I know what to do.

Acceptance Criteria:

* Minimum 3 steps
* Steps are actionable

---

### 5

As a user,
I want to paste a message for analysis,
so that I check SMS/email scams.

Acceptance Criteria:

* Text input exists
* System analyzes content

---

### 6

As a user,
I want to see a safety score,
so that I understand risk.

Acceptance Criteria:

* Score displayed (0–100)
* Updates after scan

---

### 7

As a user,
I want to see if a domain is verified,
so that I trust it.

Acceptance Criteria:

* Verified label shown
* Explanation exists

---

### 8

As a user,
I want to see scan history,
so that I track threats.

Acceptance Criteria:

* List of scans
* Status visible

---

**🟡 SHOULD**

### 9

As a user,
I want daily security tips,
so that I improve awareness.

Acceptance Criteria:

* Tip shown
* Can mark as read

---

### 10

As a user,
I want to report phishing,
so that others are protected.

Acceptance Criteria:

* Report button
* Submission stored

---

### 11

As a user,
I want to see recent scans,
so that I stay informed.

Acceptance Criteria:

* Visible on dashboard

---

### 12

As a user,
I want quick scan access,
so that I save time.

Acceptance Criteria:

* Button on home screen

---

### 13

As a user,
I want to open safe links,
so that I continue browsing.

Acceptance Criteria:

* Open button available

---

### 14

As a user,
I want to learn from past threats,
so that I improve.

Acceptance Criteria:

* History includes explanations

---
